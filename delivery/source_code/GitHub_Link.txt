KENDİN DERLEMEK İSTERSEN:

1. Visual Studio Code + PlatformIO kur
2. GitHub'dan kodu indir (Clone or Download → ZIP)
3. Zip'i aç
4. VS Code'da klasörü aç (File → Open Folder)
5. PlatformIO: Build butonuna bas
6. .pio/build/esp32-s3-devkitc-1/firmware.bin oluşacak
